# Michael Jackson | The King of Pop – Tribute Page

## Popis projektu
Tento projekt je moderní, vizuálně atraktivní a interaktivní jednostránkový web (tribute page) věnovaný legendárnímu králi popu, Michaelu Jacksonovi. Web slouží jako digitální biografie, která provází návštěvníka jeho životem, největšími kariérními milníky, ikonickou diskografií a nesmrtelným odkazem. 

Cílem bylo vytvořit prémiový uživatelský zážitek pomocí plynulých animací, dynamických efektů a čistého, luxusního designu v černo-zlaté paletě.

## Použité technologie a knihovny

### Core
- **HTML5**: Sémantická struktura stránky a SEO optimalizace.
- **Vanilla CSS3**: Pokročilý styling, responzivní layout (Flexbox/Grid), skleněný efekt (glassmorphism) a vlastní animace.
- **Vanilla JavaScript (ES6+)**: Logika ovládání menu, loaderu a interaktivních prvků.

### Knihovny a API
- **[GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)**: Použito pro plynulé úvodní animace, pohybové efekty při scrollování a komplexní časování.
- **[GSAP ScrollTrigger](https://greensock.com/scrolltrigger/)**: Zajišťuje spouštění animací přesně ve chvíli, kdy se prvek objeví na obrazovce.
- **[Particles.js](https://vincentgarreau.com/particles.js/)**: Vytváří dynamické zlaté částice v pozadí hero sekce, které reagují na pohyb myši.
- **Google Fonts**: Typografie využívající fonty *Cinzel* (nadpisy) a *Montserrat* (text), které dodávají webu elegantní nádech.

## Návod na spuštění
Projekt je postaven jako čistě statický web, takže pro jeho spuštění **není potřeba žádný build ani instalace závislostí**.

1. Stáhněte nebo naklonujte složku s projektem.
2. Otevřete soubor `Michael Jackson.html` v libovolném moderním webovém prohlížeči (Chrome, Firefox, Safari, Edge).
3. Pro plný zážitek doporučujeme stabilní internetové připojení pro načtení knihoven z CDN.

## Prvky, na které jsme nejvíce hrdí
- **Moonwalk Animation**: Interaktivní sekce "The Moonwalk", kde se pomocí GSAP animují stopy bot v nekonečné smyčce, což evokuje Michaelův ikonický taneční pohyb.
- **Dynamické počítadlo statistik**: V sekci "Síň Slávy" se při nascrollování automaticky spustí animovaný odpočet rekordů (Grammy, Guinnessovy rekordy atd.), což dodává stránce na živosti.
- **Karty diskografie**: Interaktivní karty alb s efektem převrácení (3D flip), které umožňují nahlédnout na seznam největších hitů každého alba.
- **Zlatý částicový systém**: Ambientní pozadí v úvodu, které vytváří prémiovou atmosféru hned při prvním načtení stránky.
- **Responzivní navigace**: Plně funkční mobilní menu s plynulými přechody, které zajišťuje skvělou čitelnost na všech zařízeních.
